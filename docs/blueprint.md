# **App Name**: Btruss Digital Hub

## Core Features:

- Services Overview: Showcase Btruss Services: Prominently display all the services offered by Btruss Services Pvt Ltd, including IT Services, Healthcare Services, Credit Card Sales, and Insurance Sales.
- Testimonials Section: Client Testimonials: Feature testimonials from satisfied clients to build trust and credibility.
- Inquiry Submission: Contact Form: A simple and direct contact form for potential clients to reach out.
- Responsive Layout: Mobile Responsive Design: Ensure the website is fully responsive and accessible on all devices.

## Style Guidelines:

- Primary color: A deep blue (#1A237E) to convey trust and professionalism, reflecting the company's reliability.
- Background color: A very light, desaturated blue (#E8EAF6) to provide a clean and modern backdrop that doesn't distract from the content.
- Accent color: A muted teal (#4DB6AC), for calls to action, providing a fresh and modern contrast without overwhelming the design.
- Clean and modern sans-serif font for headers and body text, ensuring readability and a professional look.
- Use a grid-based layout with plenty of white space to create a clean, organized, and visually appealing structure. The sections should easily inform and direct the user's navigation.
- Subtle transitions and hover effects on buttons and interactive elements to enhance user engagement and provide a polished feel.
